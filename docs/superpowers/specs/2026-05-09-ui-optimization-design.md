# 股票评分系统界面优化 — 设计文档

## 目标
为 v1.0 评分引擎添加简洁的 Web 界面。用户输入股票名称或代码，直接看到评分，不展示任何代码或 JSON。

## 交互流程
1. 用户打开网页，看到搜索输入框
2. 输入股票名称或代码关键词，下拉显示匹配建议
3. 选中股票后，自动请求评分并展示结果卡片

## 页面结构
- **搜索区**：输入框 + 下拉建议列表
- **结果卡片**：总分 + 等级 + 进度条 + 五维度得分条 + 形态描述

## 技术方案
- FastAPI 新增 Jinja2 模板渲染首页（`GET /`）
- 新增 `GET /api/v1/search?keyword=` 模糊搜索接口
- 复用现有 `GET /api/v1/rating/{stock_code}` 评分接口
- 前端：纯 HTML + CSS + 少量原生 JS，无框架
